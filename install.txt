* crear usuari webapp
* fer directori /home/webapp/vXXXX i entrar
* baixar otsun i otsunwebapp 
* fer virtualenv venv i activar
* fer pip install -e els dos paquets otsun i otsunwebapp
* crear fitxer /home/webapp/vXXXX/venv/lib/python3.6/site-packages/freecad.pth
i posar dins /usr/lib/freecad-python3/lib (o el que toqui)
* copiar OTSunWebApp/sample_server.py a /home/webapp/vXXXX/server.wsgi (i tunejar)
* posar dins del <VirtualHost *:80> de /etc/apache2/sites-available/000-default.conf
la configuracio:
--
   WSGIDaemonProcess webappsunscene user=webapp group=webapp python-home=/home/webapp/vXXXX/venv
    WSGIScriptAlias /webappsunscene /home/webapp/vXXXX/server.wsgi process-group=webappsunscene application-group=%{GLOBAL}
    <Directory /home/webapp/vXXXX>
        WSGIApplicationGroup %{GLOBAL}
        WSGIProcessGroup webappsunscene
        Order deny,allow
        Allow from all
        Require all granted
    </Directory>

   WSGIDaemonProcess prova user=webapp group=webapp python-home=/home/webapp/prova/venv
    WSGIScriptAlias /prova /home/webapp/prova/server.py process-group=prova application-group=%{GLOBAL}
    <Directory /home/webapp/prova>
        WSGIApplicationGroup %{GLOBAL}
        WSGIProcessGroup prova
        Order deny,allow
        Allow from all
        Require all granted
    </Directory>
--
* sudo service apache2 restart
